# Proceso Operativo

## Incorporación de nuevo boletín

1. Descargar PDF.
2. Guardar PDF en samples/.
3. Ejecutar extracción usando extraction_prompt_v1.md.
4. Validar JSON contra full_boletin.schema.json.
5. Revisar datos manualmente.
6. Cargar datos en Neon.
7. Actualizar Power BI.

## Objetivo

Mantener histórico completo de:

* empleo privado
* empleo público
* formación

## Frecuencia

Cada vez que se publique un nuevo boletín.
